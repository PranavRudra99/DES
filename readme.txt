Compilation:
cd source
make

Execution:
./MTUHash [fileName]

The MTUHash executable accepts a single optional parameter which is the HashInput file name. If no file name is specified, the default value of Hashin.txt is used. Executing MTUHash generates the output files Out1.txt and OutFinal.txt.

./AECalculator [fileName]

The AECalculator executable accepts a single optional parameter which is the HashInput file name. If no file name is specified, the default value of Hashin.txt is used. Executing AECalculator generates the output file AFR.txt.
